# 🎮 PROJECT NYZ - Fortnite Launcher (Windows Forms)

## 📋 المحتويات
لانشر احترافي لتشغيل فورتنايت سيزن 4 شابتر 2 مع واجهة Windows Forms جميلة

---

## ✨ الميزات

✅ **واجهة Windows Forms احترافية**
- تصميم Dark Theme أنيق
- ألوان سيزان (أزرق سماوي + أسود)
- أزرار جميلة وسهلة الاستخدام

✅ **تسجيل دخول سهل**
- اختيار المستخدم من قائمة
- إضافة مستخدمين جدد
- حفظ بيانات المستخدمين تلقائياً

✅ **اختيار مجلد اللعبة**
- زر "Select Folder" لاختيار مجلد اللعبة
- حفظ المسار تلقائياً
- التحقق من وجود الملف

✅ **تشغيل مباشر للعبة**
- تشغيل Fortnite من اللانشر مباشرة
- رسائل حالة واضحة
- معلومات اللعبة (السيزن والإصدار)

---

## 🚀 البدء السريع

### 1. فتح المشروع
```
1. افتح Visual Studio 2022
2. اختر: File → Open Project
3. اختر: projectnyz1.csproj
```

### 2. بناء المشروع
```
1. اضغط: Ctrl + Shift + B
2. أو: Build → Build Solution
3. انتظر حتى تكتمل عملية البناء
```

### 3. تشغيل اللانشر
```
1. اضغط: F5
2. أو: Debug → Start Debugging
3. ستفتح نافذة التسجيل
```

---

## 📝 خطوات الاستخدام

### الخطوة 1: تسجيل الدخول
```
1. اختر اسم مستخدم من القائمة (أو أضف واحد جديد)
2. أدخل كلمة المرور
3. اضغط "اختر" واختر مجلد اللعبة
4. اضغط "دخول"
```

### الخطوة 2: تشغيل اللعبة
```
1. ستظهر نافذة اللعبة
2. معلومات اللعبة تظهر تلقائياً
3. اضغط "تشغيل اللعبة"
4. الفورتنايت ستفتح تلقائياً
```

### الخطوة 3: الخروج
```
1. اضغط "الخروج" للعودة إلى التسجيل
2. أو أغلق النافذة مباشرة
```

## ⚠️ الأخطاء الشائعة

### ❌ "Build failed"
**الحل:**
- تأكد من تثبيت .NET 6.0
- احذف مجلدات: bin, obj, .vs
- أعد فتح المشروع

### ❌ "لم يتم العثور على اللعبة"
**الحل:**
- تأكد من وجود مجلد Fortnite
- تحقق من المسار الصحيح
- جرب نسخة مختلفة من الفورتنايت

### ❌ "مجلد اللعبة غير موجود"
**الحل:**
- أعد اختيار المجلد
- تأكد من حفظ المسار
- جرب نسخ المسار يدوياً
